# Desafio-front-end

* Este projeto foi feito em Vue e Nuxt seguindo as normas do desafio e implementadas as suas 3 etapas. 
* Foi hospedado no firebase com o link [desafio-front-end](https://desafio-front-end-989ff.firebaseapp.com/)

# Instalação do projeto

* yarn install

# Subir o projeto 

* *yarn run dev

# Build do projeto

* *yarn run build

# Deploy para o firebase

* firebase deploy
